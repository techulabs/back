HTEX01 - Formulario con etiquetas HTML5 - placeholder, required

Vamos a crear una página básica HTML en la que probaremos etiquetas y atributos de interés. El ejercicio se puede hacer en la página JSFiddle

Empezaremos creando la página HTML tal que: 

<!doctype html>
<h1>¡Hola Mundo!</h1>

Añadimos un título a la página.

Añadimos también un formulario y un input de texto, al que añadimos una id para identificarlo.

Completamos el formulario añadiendo un botón para enviarlo.

<!doctype html>
<title>This is a test</title>
<h1>¡Hola Mundo!</h1>
<form>
  <input id="formNombre" type="text"/>
  <br />
  <button>Enviar</button>
</form>

En este punto podemos comprobar que, si le damos al botón enviar, se envía el formulario, tenga o no valor el input de texto.

Si añadimos al input del formulario el atributo required, no podremos enviarlo a no ser que le demos un valor.

<!doctype html>
<title>This is a test</title>
<h1>¡Hola Mundo!</h1>
<form>
  <input id="formNombre" type="text" required />
  <br />
  <button>Enviar</button>
</form>

El atributo placeholder nos ayuda a identificar qué ha de contener un input de texto.

<!doctype html>
<title>This is a test</title>
<h1>¡Hola Mundo!</h1>
<form>
  <input id="formNombre" type="text" required placeholder="Nombre" />
  <br />
  <button>Enviar</button>
</form>

Hay tipos de input más específicos, como por ejemplo email, que añaden una validación previa al envío, comprobando que el contenido cumpla un patrón predeterminado.

<!doctype html>
<title>This is a test</title>
<h1>¡Hola Mundo!</h1>
<form>
  <input id="formNombre" type="text" required placeholder="Nombre" />
  <br />
	<input id="formEmail" type="email" placeholder="email" />
	<br />
  <button>Enviar</button>
</form>

Existe también el input de tipo number que cuenta con atributos específicos, min y max, para delimitar los números permitidos.

<!doctype html>
<title>This is a test</title>
<h1>¡Hola Mundo!</h1>
<form>
  <input id="formNombre" type="text" required placeholder="Nombre" />
  <br />
	<input id="formEmail" type="email" placeholder="email" />
	<br />
	<input id="formAge" min="18" max="99" type="number"/>
	<br />
  <button>Enviar</button>
</form>

También podemos añadir etiquetas a los input para mejorar su usabilidad.

<!doctype html>
<title>This is a test</title>
<h1>¡Hola Mundo!</h1>
<form>
	<label>Nombre</label>
  <input id="formNombre" type="text" required placeholder="Nombre" />
  <br />
	<label>Email</label>
	<input id="formEmail" type="email" placeholder="email" />
	<br />
	<label>Edad</label>
	<input id="formAge" min="18" max="99" type="number"/>
	<br />
  <button>Enviar</button>
</form>

Existen más tipos de input, como por ejemplo date y color

<!doctype html>
<title>This is a test</title>
<h1>¡Hola Mundo!</h1>
<form>
  <label>Nombre</label>
  <input id="formNombre" type="text" required placeholder="Nombre" />
  <br />
	<label>Email</label>
	<input id="formEmail" type="email" placeholder="email" />
	<br />
	<label>Edad</label>
	<input id="formAge" min="18" max="99" type="number"/>
	<br />
	<input id="formDate" type="date"/>
	<br />
	<input id="formColor" type="color"/>
	<br />
  <button>Enviar</button>
</form>
