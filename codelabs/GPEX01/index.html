GPEX01 - Comprobar la instalación de IntelliJ IDEA

En este codelab nos familiarizaremos con el entorno de trabajo de IntelliJ, un IDE especializado para desarrollo de aplicaciones empresariales basadas en Spring. En el Menú File del IDE pulsar sobre la opción Settings...:

  • Appeareance & Behavior
  • Keymap
  • Editor
  • Plugins
  • Version Control

Explorar el Project Structure (Project settings, Sources, Paths, Dependencies...) Ctrl + Alt + Shift + S

Stop. Hitting. Ctrl+S.
